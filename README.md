# GeniusBase
Genius base is a simple to use wiki web application, the main features are

- Easy to create article
- Organise the artilce by category and tags
- Smart search functionality to find an article quick and easy


# Technology
- .Net core
- Code first
- Lucene .net
- Serilog
- Automapper
- Angularjs
- Bootstrap 4
- SCSS
